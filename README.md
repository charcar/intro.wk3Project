# Ping Pong
#### A game for all ages
### 1/22/2016

#### By Charles Carter

## Description

This application accepts an integer input and...
* returns 'ping' if divisible by 3
* returns 'pong' if divisible by 5
* returns 'pingpong' if divisible by 15
